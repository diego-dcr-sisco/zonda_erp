# SISCO-ERP

SISCO-ERP es un sistema de planificación de recursos empresariales (ERP) diseñado para SISCOPLAGAS y empresas dedicadas al manejo integral de plagas . Permite gestionar inventarios, ventas, compras, clientes, proveedores y reportes financieros desde una sola plataforma.

## Características principales

- Gestión de inventario y productos
- Control de ventas y compras
- Administración de clientes y proveedores
- Generación de reportes financieros
- Interfaz intuitiva y fácil de usar
